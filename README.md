# Simulador de Producción Industrial

Aplicación web interactiva desarrollada como proyecto académico para la asignatura de programación web.

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Descripción
La aplicación permite simular un proceso productivo industrial analizando la relación entre producción, demanda e inventarios.
